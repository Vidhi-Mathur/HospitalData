
## Installation


1. Clone the repository:
   ```bash
   git clone https://github.com/Vidhi-Mathur/HospitalData.git
    ```
2. Navigate to project repository:
    ```bash
   cd HospitalData
    ```
3. Install frontend dependencies: 
   ```bash
   npm install
    ```
4. Start the vite server
    ```bash
   npm run dev
